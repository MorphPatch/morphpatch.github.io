# MorphPatch

Project page for **MorphPatch: Enhancing VR Interaction on Shape Displays using Surface Approximation and Visuo-Haptic Illusions**.

Published at [https://morphpatch.github.io](https://morphpatch.github.io).

## Publish

In **Settings → Pages**, deploy from the `main` branch at `/ (root)`.

```bash
cd /Users/yingw/Desktop/morphpatch.github.io
git remote add origin https://github.com/MorphPatch/morphpatch.github.io.git
git push -u origin main
```
